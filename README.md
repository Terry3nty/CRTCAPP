# CRTC App (Official Distribution)

This is the official distribution repository for the CRTC App. 
The source code is maintained in a private repository for security and proprietary reasons.

## 📥 Download Latest Version
Go to the [Releases](https://github.com/Terry3nty/CRTCAPP/releases) section to download the latest .apk file for Android.

## 🛠 Features
* Daily passages
* Sunday school
* Daily streaks

## 📝 Installation Guide
1. Download the APK from the link above.
2. Open the file on your Android device.
3. If "Play Protect" blocks it, click "Install anyway". 
4. Launch and enjoy!
